Подключение Яндекс Диска по протоколу WebDAV

  • Необходимое время: 10 мин.;
  • Сложность: средняя;
  • Необходимые навыки: браузер, мышь, клавиатура.

Что такое WebDAV?

  • Набор расширений и дополнений к протоколу HTTP;
  • Поддерживает выполнение основных файловых операций над объектами на удаленном сервере;
  • Поддерживает выполнение расширенных файловых операций (блокировки, поддержка версий);
  • Поддерживает работу с любым типом объектов (не только файлы);
  • Поддерживает работу с метаданными;
  • Поддерживает одновременную работу над несколькими объектами.

Как подключить

  • Зарегистрируйте аккаунт Яндекс ID по инструкции
  • Создайте пароль по инструкции «Пароли приложений»
  • На ПК подключите Яндекс Диск в качестве сетевого диска

Особенности подключения в разных ОС

  • Windows 11
  • Windows 10
  • Linux
  • Mac Os

Ограничение. При установке в ОС Windows может быть запущен только Защитник Windows! Никаких сторонних антивирусов!
